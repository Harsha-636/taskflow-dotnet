# ⚡ TaskFlow — Enterprise Task Management System

Full-stack enterprise web app built with **.NET 8 Web API (C#)**, Entity Framework Core, JWT Auth, SQLite and AI task suggestions.

## 🛠️ Tech Stack
- C# / ASP.NET Core 8 Web API
- Entity Framework Core 8 + SQLite
- JWT Bearer Authentication
- BCrypt Password Hashing
- Swagger/OpenAPI
- Vanilla JS + HTML5 + CSS3
- Docker + Render deployment

## 🚀 Run Locally
```bash
dotnet restore
dotnet run
```
Open http://localhost:5000
